what is fenn

fenn is a fast, minimal browser for mac, ios, android, and windows. no bloat, just results.

installation

2. run it
3. open fenn and browse

fenn for iOS is available as a sideloadable IPA. requires iOS 14.0+

download IPA

sideloading instructions

option 1: AltStore
1. install altstore from altstore.io
2. download the ipa above
3. open altstore and install

option 2: Sideloadly
1. download sideloadly from sideloadly.io
2. connect iphone via usb
3. drag ipa into sideloadly
4. sign with apple id

option 3: TrollStore
tap the ipa to install (if jailbroken)

fenn for android is available as a free APK. requires android 7.0+

download APK

install instructions

1. download the APK above
2. open settings → security
3. enable "install from unknown sources"
4. open the downloaded APK file
5. tap install

no account needed. no expiry. just install and go.

fenn for linux is available as an AppImage. works on most distributions

download AppImage

• make it executable: chmod +x fenn.AppImage
• run: ./fenn.AppImage

fenn for windows is available as an installer. requires windows 10 or 11

download installer
